🛡️ Secure KeyLogger
Secure KeyLogger for Digital Auditing & Recovery

A lightweight Python-based tool designed to log keystrokes for personal data recovery, digital forensics, or system auditing.

🚀 Features
Real-time Logging: Captures keystrokes accurately.

Stealthy Execution: Runs efficiently in the background.

Formatted Output: Saves logs in a readable format for easy auditing.

Local Storage: Keep your data on your machine (no cloud required).

🛠️ Installation
Clone the repository:

Bash
git clone https://github.com/Rajat-sde/keyLogger.git
cd keyLogger
Install dependencies:
This project uses pynput. Install it via pip:

Bash
pip install pynput
💻 Usage
Run the main script to start the logger:

Bash
python keystrokeMain.py
The script will begin recording keystrokes to a local file (e.g., log.txt).
